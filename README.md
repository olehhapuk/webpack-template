# Webpack template

(Pug, Scss, Babel)

### Description

This template builds pug files from views folder to dist and automatically injects script from js folder with same name as .pug file.
If you want to include css to your layout import .scss file in corresponding .js file and it will be automatically injected into resulting .html file.

### Scripts

- Run dev server: `npm run dev`
- Build project to dist folder: `npm run build`

### First steps

Create .env file in root directory using .env.example template.
Assign NODE_ENV value of development or production.
